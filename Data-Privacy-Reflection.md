What types of data are considered confidential at focus bear?


At focus bear, confidential data is basically all information that is related to users, employees and the business that is not meant to be disclosed to the public.

Examples of confidential data include:

Any personal information such as names, account details  and email addresses
Any information pertaining to habit and routine information That is input by users
Any data that shows app usage and analytics information about how users interact with the focus bear app
Any information pertaining to payment and subscription
Any A.I related data 
All internal company documentation 


User data is the most sensitive category.  The privacy policy explicitly states that the company collects account and usage information in order to improve the product. Any data about user habits is encrypted.  Any team member at focus bear must not access user habit data in the event a user reports an issue and has asked for help. As an intern for focus bear, all information pertaining to user data and internal company materials is confidential and should only be accessed by asking for consent. This information should never be shared with anyone from outside the organisation.



What are best practices for handling of confidential data?

All internal company information should be kept 100% private and confidential. When dealing with confidential data, it is important to keep it secure and only share it with team members when authorised. 
Any sensitive data should not be downloaded on an unsecured device. This type of data should also not be copied into search engines, A.I tools, free file sharing sites, cloud based storage accounts or any other types of public messaging forums or platforms. 
It is also important to use strong passwords, use two factor authentication and use secure file sharing tools that can be used to protect company and user data.
